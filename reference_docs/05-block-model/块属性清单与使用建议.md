# 块属性清单与使用建议

- 适用版本：SiYuan `v3.5.7`
- 官方仓库同步到：`siyuan-note/siyuan@master` + Release `v3.5.7`（2026-02-14）
- 最后核对：2026-02-21
- 稳定性：stable（含部分约定属性）
- 权威来源：
  - <https://github.com/siyuan-community/siyuan-developer-docs/tree/main/docs/zh-Hans/reference/block>
  - <https://github.com/siyuan-note/siyuan/blob/master/API_zh_CN.md>

## 1. 使用原则

- 自定义属性必须使用 `custom-` 前缀。
- 属性值保持轻量，只存业务状态，不存大文本。
- 读取外部属性前先判空，避免主题或插件缺失导致异常。

## 2. 常见属性清单

|属性名|适用块|含义|注意事项|
|---|---|---|---|
|`custom-avs`|全部|关联的属性视图 ID 列表|版本差异需核对|
|`custom-data-assets`|挂件块|挂件资源路径|与 `data-assets` 等价|
|`custom-heading-mode`|嵌入块|是否隐藏标题下方内容|`0` 显示，`1` 隐藏|
|`custom-hidden`|文档块|文档树中隐藏|仅影响展示|
|`custom-liandi-articleId`|文档块|链滴文章 ID|第三方集成|
|`custom-reminder-wechat`|全部|微信提醒时间|依赖外部服务|
|`custom-riff-decks`|全部|闪卡卡包 ID 列表|多值|
|`custom-sy-fullwidth`|文档块|文档全宽显示|`true`/`false`|
|`custom-sy-readonly`|文档块|文档只读|谨慎使用|
|`custom-dailynote-{yyyymmdd}`|文档块|日记标识|键名含日期|
|`data-assets`|挂件块|挂件引用资源路径|导出相关|
|`data-export-html`|挂件块|导出替换 HTML|导出相关|
|`data-export-md`|挂件块|导出替换 Markdown|导出相关|

## 3. 设计建议

- 业务状态放属性，正文与结构放块内容。
- 统一命名空间，例如 `custom-myplugin-*`。
- 对外部属性建立兼容层，避免直接依赖单一值。

## 4. 本章如何使用

- 设计数据模型时先确认“属性还是块内容”。
- 需要批量读写属性时优先走 `/api/attr/*`。
- 遇到不明属性先查 `attributes` 表与官方文档。
