## v0.0.9

首个发布版本，提供两大类功能：

一、文档操作类功能

1. 反链与正链文档导出
   支持导出当前文档、打包导出反链文档、打包导出正链文档；可选择包含媒体资源，便于归档与分享。
2. 反链/子文档列表插入
   可将当前文档的反链文档或子文档列表一键插入到正文中，自动生成可点击的文档链接。
3. 反链文档移动为子文档
   批量把反链文档移动到当前文档之下，作为子文档进行层级归类（桌面端）。
4. 重复文档识别与清理
   基于标题相似度识别同层级重复文档，集中查看并选择删除。
5. 清理空段落
   扫描并移除当前文档中的多余空段落，保持内容整洁。

二、关键信息筛选功能

1. 关键内容自动提取
   从文档中提取标题、加粗、斜体、高亮、备注、标签等关键内容，集中展示在侧栏面板。
2. 点击跳转定位
   点击关键内容条目可在当前文档中平滑滚动定位到对应块。
